# WAV-NCM-MP3-Audio-ConverterBOX
是一款基于 Python 开发的本地化、轻量级音频处理工作站。它集成了 FFmpeg 引擎与 NCM 解密算法，旨在为用户提供音质良好，完全免费！的音频转换解决方案



核心功能 / Key Features

高保真格式转换 (High-Fidelity Conversion)
支持 WAV 与 MP3 之间的双向无损/高品质转换。
默认输出 320kbps 发烧级比特率，确保极致听感。

Supports bidirectional conversion between WAV and MP3.
Hardcoded 320kbps bitrate output to ensure audiophile-grade sound quality.


NCM 专项解密 (NCM Decryption)

基于 AES 密码学算法，一键提取网易云音乐加密包内的原始音频。
支持自动识别并剥离加密壳，恢复 MP3/FLAC 原文件。

Utilizes AES cryptographic algorithms to extract original audio from encrypted .ncm files.
Automatically identifies and strips encryption layers to recover source MP3/FLAC files.
